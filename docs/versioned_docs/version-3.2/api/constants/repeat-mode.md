# Repeat Mode

All RepeatMode types are made available through the named export `RepeatMode`:

```ts
import { RepeatMode } from 'react-native-track-player';
```

| Name | Description |
|------|-------------|
| `Off` | Doesn't repeat. |
| `Track` | Loops the current track. |
| `Queue` | Repeats the whole queue. |
